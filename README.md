<details>
  <summary align="center"><b>Contribution Snake Setup Details 🐍</b></summary>
  <br>
  To enable the contribution snake animation at the top, you'll need to set up a GitHub Action.
  Create <code>.github/workflows/snake.yml</code> in your repository with the following content:
  <pre>
name: Generate Snake
on:
  schedule:
    - cron: "0 */12 * * *"
  workflow_dispatch:
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: ${{ github.repository_owner }}
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
      - uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
  </pre>
  Once the action runs, the snake animation will automatically appear in your README.
</details>
